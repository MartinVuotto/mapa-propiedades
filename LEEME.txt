CÓMO ACTUALIZAR EL PROYECTO
-----------------------------
Desde la terminal en VS Code:
  git add .
  git commit -m "descripción del cambio"
  git push

Los cambios se reflejan en la URL pública en 1-2 minutos.

CÓMO IMPORTAR PROPIEDADES DESDE EXCEL
---------------------------------------
Columnas requeridas: zona, barrio, tipo, precio_usd, expensas_ars,
m2_totales, m2_cubiertos, ambientes, nombre, url
Usar el botón "Importar Excel" en el header de la aplicación.

STACK
-----
- Frontend: HTML + CSS + JS puro
- Base de datos: Firebase Realtime Database (plan gratuito)
- Hosting: GitHub Pages (gratuito)
- Mapa: Leaflet.js + CartoDB tiles
- Excel: SheetJS
